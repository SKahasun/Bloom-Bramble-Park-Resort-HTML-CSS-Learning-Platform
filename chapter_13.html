CSS Colors

1. Color Names

CSS supports a wide range of color names that can be used to specify colors in your stylesheets. These color names are predefined and recognized by all modern browsers. For example, you can use color names like "red", "blue", "green", "yellow", "cyan", "magenta", "orange", and many more to set the color of text, backgrounds, and other elements on a webpage. Using color names can make your CSS code more readable and easier to understand.

I am colored using 'tomato'.

I am colored using 'slateblue'.

Code:

.name {
  background-color: tomato;
  color: white;
}

.name-2 {
  background-color: slateblue;
  color: gold;
}

2. Hex Colors

Hexadecimal (hex) colors are a way to represent colors in CSS using a six-digit code. The code consists of three pairs of hexadecimal digits, where each pair represents the intensity of red, green, and blue (RGB) components of the color. For example, the hex code "#f03c14" represents a shade of red, where "f0" is the intensity of red, "3c" is the intensity of green, and "14" is the intensity of blue. Hex colors are widely used in web design and allow for a vast range of color options.

I am colored using '#f03c14'.

I am colored using '#bc0d0d'.

Code:

.hex {
  background-color: #f03c14;
  color: white;
}

.hex-2 {
  background-color: #bc0d0d;
  color: white;
}

3. RGB Colors

RGB (Red, Green, Blue) colors are a way to specify colors in CSS using the RGB color model. In this model, colors are defined by their red, green, and blue components, each of which can have a value ranging from 0 to 255. For example, the RGB color "rgb(227, 36, 36)" represents a shade of red where the red component has an intensity of 227, the green component has an intensity of 36, and the blue component has an intensity of 36. RGB colors can also be specified using RGBA (Red, Green, Blue, Alpha) format, which includes an additional alpha channel for transparency.

I am colored using 'rgb(227, 36, 36)'.

I am colored using 'rgba(236, 8, 11, 0.977)'.

Code:

.rgb {
  background-color: rgb(227, 36, 36);
  color: white;
}

.rgba {
  background-color: rgba(236, 8, 11, 0.977);
  color: white;
}

4. Grey Colors

Grey colors in CSS can be created using various methods, including color names, hex codes, and RGB values. For example, you can use the color name "lightgrey" to create a light grey color, or you can use the hex code "#eeeeee" for a similar effect. Additionally, you can specify grey colors using RGB values where the red, green, and blue components are equal, such as "rgb(60, 60, 60)" for a darker grey. Grey colors are often used in web design for backgrounds, borders, and text to create a neutral and balanced look.

I am colored using 'lightgrey'.

I am colored using 'grey-dark'.

Code:

.grey-light {
  background-color: #eeeeee;
  color: #333333;
}

.grey-dark {
  background-color: #3c3c3c;
  color: white;
}
