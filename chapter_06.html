1. Audio

The <audio> element is used to embed sound content in a web page. It supports various audio formats and provides controls for playback.

Code:

<audio controls muted autoplay loop>
  <source src="Audio/universfield-ringtone.mp3" type="audio/mpeg" />
</audio>

2. Video

The <video> element is used to embed video content in a web page. It supports various video formats and provides controls for playback.

Code:

<video width="800" controls muted autoplay loop>
  <source src="Video/music pad.mp4" type="video/mp4" />
</video>
