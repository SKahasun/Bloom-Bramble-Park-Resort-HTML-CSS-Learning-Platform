1. Text Rendering

Text rendering properties in CSS allow you to control how text is displayed on a webpage. For example, the <text-rendering> property can be used to optimize the legibility or speed of text rendering, while the <font-smoothing> property can be used to control the anti-aliasing of text. These properties can help improve the readability and visual appeal of text on a webpage.

this text is transformed to capitalize.

2. Text Spacing

Text spacing properties in CSS allow you to control the spacing between characters, words, and lines of text. For example, the <letter-spacing> property can be used to adjust the spacing between characters, while the <word-spacing> property can be used to adjust the spacing between words. The <line-height> property can be used to control the spacing between lines of text. These properties can help improve the readability and visual appeal of text on a webpage.


Letter-Spacing

This text has increased letter spacing.


Word-Spacing

This text has increased word spacing.


Line-Spacing

This text has increased line height. The spacing between the lines of text is adjusted to make it more readable and visually appealing. Line height is especially useful when dealing with large blocks of text.

Code:

.letter-spacing {
  letter-spacing: 2px; 
  border: 1px solid red;
  margin: 10px;
}

.word-spacing {
  word-spacing: 10px; 
  border: 1px solid red;
  margin: 10px;
}

.line-height {
  line-height: 3; 
  border: 1px solid red;
  margin: 10px;
}

3. Text Position

Text position properties in CSS allow you to control the alignment and indentation of text. For example, the <text-align> property can be used to align text to the left, right, center, or justify it across the width of a container. The <text-indent> property can be used to create an indentation at the beginning of a paragraph. These properties can help improve the readability and visual appeal of text on a webpage.



Text Align


Justify

This text is justified. Justified text is spaced so that the left and right sides of the text block both have a clean edge. This is often used in newspapers and books.


Center

This text is centered. Centered text is aligned to the middle of the container. This is useful for headings or emphasizing important information.


Right Align

This text is aligned to the right. Right-aligned text is aligned to the right edge of the container. This can be useful for specific design requirements.


Text Indent

This text has an indentation of 50 pixels. Indenting text is often used in paragraphs at the beginning of a section or for quotations.

Code:

.line-height {
  line-height: 3; 
  border: 1px solid red;
  margin: 10px;
}
.text-align-justify {
  text-align: justify;
  border: 1px solid red;
  margin: 10px;
  padding: 10px;
}

.text-align-center {
  text-align: center;
  border: 1px solid red;
  margin: 10px;
  padding: 10px;
}

.text-align-right {
  text-align: right;
  border: 1px solid red;
  margin: 10px;
  padding: 10px;
}

.text-indent {
  text-indent: 50px; 
  border: 1px solid red;
  margin: 10px;
  padding: 10px;
}

4. CSS Fonts

CSS font properties allow you to control the appearance of text on a webpage. For example, the <font-family> property can be used to specify the font type, while the <font-size> property can be used to adjust the size of the text. The <font-weight> property can be used to make text bold or lighter, and the <font-style> property can be used to italicize text. These properties can help improve the readability and visual appeal of text on a webpage.
