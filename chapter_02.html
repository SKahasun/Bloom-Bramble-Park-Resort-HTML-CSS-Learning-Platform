1. Heading

My First h1

My Second h2

My Fourth h4

My Fifth h5
My Sixth h6

Code: 

<h1>My First h1</h1>
<h2>My Second h2</h2>
<h4>My Fourth h4</h4>
<h5>My Fifth h5</h5>
<h6>My Sixth h6</h6>

2. Paragraph

The paragraph element is used to define a paragraph in HTML.

Code: 

<p>
  Welcome to my Webpage.
</p>

3. Hyperlink

The hyperlink element is used to create a link to another webpage or file. my GitHub Profile

Code: 

<a href="https://github.com/SKahasun">my GitHub Profile</a>

4. Image

The image element is used to embed an image in an HTML page. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for the image if it cannot be displayed.

Code: 

<img src="Image/stasknop.jpg" alt="stasknop image" width="200" />

5. Image Map

The image map element is used to define clickable areas on an image. The map element contains one or more area elements that define the clickable areas.

Code: 

<img src="/Image/tankilevitch.jpg" usemap="#polaroid-map" alt="Three Polaroids on 
a white woodensurface" width="1024" height="682">

<map name="polaroid-map">
    <area shape="rect" coords="148,235,373,488" alt="Left Polaroid" 
href="https://images.pexels.com/photos/6024723/pexels-photo-6024723.jpeg">
    
    <area shape="rect" coords="425,232,653,485" alt="Middle Polaroid" href="
https://images.pexels.com/photos/6024682/pexels-photo-6024682.jpeg">
    
    <area shape="rect" coords="684,228,910,481" alt="Right Polaroid" href="
https://images.pexels.com/photos/6024717/pexels-photo-6024717.jpeg">
</map>

6. List

The list element is used to create ordered and unordered lists in HTML.
Ordered List:

  1. First Item
  2. Second Item
  3. Third Item
Unordered List:
  • Apple
  • Banana
  • Orange

Code: 

<strong>Ordered List:</strong>
<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

<strong>Unordered List:</strong>
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>

7. Table

The table element is used to create tables in HTML.
Table:

Name Age City
Ahmed 25 Dhaka
Rahim 30 Chittagong

Code: 

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Ahmed</td>
    <td>25</td>
    <td>Dhaka</td>
  </tr>
  <tr>
    <td>Rahim</td>
    <td>30</td>
    <td>Chittagong</td>
</table>

8. Break

The break element is used to insert a line break in HTML.
Example:
This is the first line.
This is the second line.
This is the third line.

Code: 

<br>
This is the first line.<br>
This is the second line.<br>
This is the third line.

9. Progress Bars

The progress element is used to create a progress bar in HTML. The value attribute specifies the current value of the progress bar, and the max attribute specifies the maximum value of the progress bar.
Example:

Code: 

<progress value="70" max="100"></progress>

10. Computer Code

The code element defines computer code. It will be displayed in the default monospace font of the browser.

Code: 

<code>It will be displayed in the default monospace font of the browser.</code>

11. Iframes

The iframe element is used to embed another HTML page within the current page. The src attribute specifies the URL of the page to be embedded.
Example:

Code: 

<iframe src="/HomePage.html" width="800" height="600"></iframe>

12. HTML Entities

HTML entities are used to display special characters in HTML. For example, the & entity is used to display the ampersand (&) character, and the < entity is used to display the less-than (<) character.

This is a greater than symbol: >

This is a less than symbol: <

This is an ampersand: &

This is a copyright symbol: © 2025

This is a registered trademark symbol: ®

Currency symbols: Dollar ($), Euro (€), Pound (£), Yen (¥)

Mathematical symbols: Plus-Minus (±), Multiplication (×), Division (÷)

Code: 

<p>This is a greater than symbol: &gt;</p>
<p>This is a less than symbol: &lt;</p>
<p>This is an ampersand: &amp;</p>
<p>This is a copyright symbol: &copy; 2025</p>
<p>This is a registered trademark symbol: &reg;</p>
