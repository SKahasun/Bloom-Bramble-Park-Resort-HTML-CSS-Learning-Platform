1. Universal Selector

The universal selector (*) selects all elements in the document.

All div are apply with Uneversal Selector

Code: 

* {
        margin: 0;
        padding: 0;
        box-sizing: border-box;
      }

2. Type Selector

The type selector (also called element selector) selects all elements of a specific type.


All h1 heading is apply in Type Selector. Which is aquamarine background colour.

4. Class Selector, Multiple Classes, CSS Pseudo-Element

The class selector uses the class attribute of an HTML element to select all elements with a specific class.
Multiple classes can be applied to an element by separating them with spaces in the class attribute.
CSS pseudo-elements allow you to style specific parts of an element, such as the first letter or the first line of text.

5. CSS Pseudo-Element

CSS pseudo-elements allow you to style specific parts of an element, such as the first letter or the first line of text.


::first-letter and ::first-line pseudo-elements.

Improtantce of learning CSS

Learning CSS (Cascading Style Sheets) is important for several reasons:
Visual Design: CSS allows you to style and format the appearance of your web pages. You can control colors, fonts, layouts, and more, making your website visually appealing and user-friendly.

Separation of Content and Style:
CSS helps separate the content (HTML) from the presentation (CSS). This makes your code more organized and easier to maintain. You can apply consistent styles across multiple web pages by linking a single CSS file.

Code: 

.fistlater::first-letter {
  font-size: 30px;
  color: black;
}

.fistlater::first-line {
  /* font-size: 40px; */
  background-color: red;
  color: white;
}

6. CSS Attribute Selector

CSS attribute selectors allow you to select elements based on the presence or value of their attributes.
They provide a powerful way to target specific elements without needing to add extra classes or IDs.

[attribute] Selector


[attribute="value"]


[attribute|="value"]

This paragraph has the lang attribute exactly equal to "en".

This paragraph has the lang attribute starting with "en-" (e.g., "en-US").

This paragraph does not match the selector as its lang value is "fr".


[attribute~="value"]

This paragraph has the lang attribute starting with "en-" (e.g., "en-US").

[attribute^="value"]

Dotted border style is applyed by [attribute^="value"] selector

Code: 

[placeholder] {
  border: 3px solid red;
  margin-bottom: 5px;
}

[type="submit"] {
  background-color: green;
  color: white;
}

[type="reset"] {
  background-color: red;
  color: white;
}

[lang|="en"] {
  border: 2px solid red;
  padding: 10px;
  margin-bottom: 5px;
}

[lang~="US"] {
  background-color: red;
  color: white;
}

[lang^="en"] {
  border-style: dotted;
}
