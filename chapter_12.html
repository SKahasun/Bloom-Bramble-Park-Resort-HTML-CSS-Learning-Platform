1. Box Sizing Types

The box-sizing property in CSS determines how the total width and height of an element are calculated. There are two main values for this property: content-box and border-box. By default, elements use the content-box value, which means that the width and height properties only apply to the content area of the element, excluding padding and borders. On the other hand, when using border-box, the width and height properties include the content, padding, and borders. This makes it easier to manage layouts, as the total size of the element remains consistent regardless of padding or border changes.

I am Content Box (Wider)

I am Border Box (Exact Width)

Code:

/* Total Width = 200px + 40px (padding) + 20px (border) = 260px */
.content-box {
  width: 200px;
  padding: 20px;
  border: 10px solid red;
  box-sizing: content-box;
}

/* Total Width = 200px (Padding and Border are included inside) */
.border-box {
  width: 200px;
  padding: 20px;
  border: 10px solid red; 
  box-sizing: border-box;
}

2. CSS Margins

Margins in CSS are used to create space around elements, outside of any defined borders. The margin property can be set to a specific value (e.g., 20px) or to auto, which allows the browser to calculate the margin based on the available space. When using auto, it can center an element horizontally within its container if the left and right margins are set to auto. Additionally, when two vertical margins meet, they can collapse into a single margin that is equal to the larger of the two, which is known as margin collapsing.

I have 50px margin on all sides.

I am centered horizontally.

Box 1 (Bottom Margin: 50px)

Box 2 (Top Margin: 30px)

Code:

.margin-standard {
  margin: 50px;
}

.margin-auto {
  width: 50%;
  margin: 0 auto;
}

.box1 { margin-bottom: 50px; }
.box2 { margin-top: 30px; }
/* The gap between them is 50px, NOT 80px */

3. CSS Outline

The CSS outline property is used to draw an outline around an element, outside of its border. Unlike borders, outlines do not take up space in the layout and do not affect the size of the element. Outlines are typically used for accessibility purposes, such as indicating focus on interactive elements like buttons or links.

I have a black border and a red outline.

Code:

.outline-example {
  border: 5px solid black;
  outline: 5px dashed red;
  outline-offset: 5px;
}

4. CSS Borders

The CSS border property is used to define the border around an element. It consists of three components: border-width, border-style, and border-color. The border-width specifies the thickness of the border, the border-style defines the pattern (e.g., solid, dashed, dotted), and the border-color sets the color of the border. Borders can be applied to all sides of an element or to specific sides using properties like border-top, border-right, border-bottom, and border-left.

I have a double red border.

I have rounded corners with a radius of 20px.

Code: 

.border-style {
  border: 5px double red;
}

.border-radius {
  border: 2px solid red;
  border-radius: 20px;
}

5. CSS Padding

Padding in CSS is the space between the content of an element and its border. The padding property can be set to a specific value (e.g., 20px) or to different values for each side (padding-top, padding-right, padding-bottom, padding-left). Padding increases the total size of the element, as it adds space inside the border.

I have 50px of padding on all sides.

Code:

.padding-example {
  border: 1px solid black;
  padding: 50px;
  background-color: red;
}

6. CSS Width and Height

The width and height properties in CSS are used to set the dimensions of an element. They can be defined using fixed units (e.g., pixels) or relative units (e.g., percentages). When using fixed units, the element will have a specific size regardless of the viewport or container size. When using percentages, the size of the element will be relative to its parent container, allowing for responsive designs that adapt to different screen sizes.

I have a fixed width of 300px and height of 100px.

I have a width of 50% and height of 50px.

Code:

.dimension-fixed {
  width: 300px;
  height: 100px;
}
.dimension-percent {
  width: 50%;
  height: 50px;
}
