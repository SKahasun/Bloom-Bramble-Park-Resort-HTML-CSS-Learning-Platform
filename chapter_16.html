1. Display Vs Visibility

In HTML5, semantic elements are used to give meaning to the content they contain. These elements help improve the structure and accessibility of a webpage by clearly defining the purpose of each section of content. Semantic elements can be broadly categorized into three main types: structural elements, layout elements, and formatting elements.

  visibility: hidden;

2. CSS Object Fit

CSS Object Fit is a property that allows you to control how an image or video fits within its container. It provides several values such as contain, cover, fill, and none, which determine how the media should be resized and cropped to fit the container. This property is particularly useful for maintaining the aspect ratio of images and videos while ensuring they fit within their designated space on a webpage.

Code:

.fit-contain { object-fit: contain; }
.fit-cover { object-fit: cover; }
.fit-fill { object-fit: fill; }

3. Cursor

The CSS Cursor property is used to specify the type of cursor that should be displayed when a user hovers over an element on a webpage. It allows web developers to enhance user experience by providing visual feedback based on the context of the interaction. Common cursor values include pointer, default, text, not-allowed, and grab, among others, each indicating a different type of interaction or functionality associated with the element.

4. Over Flow

The CSS Overflow property is used to control how content that exceeds the dimensions of its container is handled on a webpage. It allows developers to specify whether the overflowed content should be visible, hidden, scrollable, or automatically managed based on the content size. This property is essential for maintaining a clean and organized layout, especially when dealing with dynamic content or responsive designs.

5. CSS Opacity

CSS Opacity is a property that allows you to control the transparency of an element on a webpage. It accepts values between 0 and 1, where 0 means completely transparent and 1 means fully opaque. This property is useful for creating visual effects, such as fading elements in or out, layering content, or adding emphasis to certain parts of a webpage while keeping others less prominent.

Opacity: 1 (Fully Opaque)

Opacity: 0.5 (Semi-Transparent)

Opacity: 0.25 (More Transparent)

Code:

.op-100 { opacity: 1; }
.op-50 { opacity: 0.5; }
.op-25 { opacity: 0.25; }

6. CSS Shadow

CSS Shadow is a property that allows you to add shadow effects to elements on a webpage. It includes properties like text-shadow for adding shadows to text and box-shadow for adding shadows to boxes or other elements. These properties enable developers to create depth, highlight important content, and enhance the overall visual appeal of a webpage by simulating light and shadow effects.

Text Shadow Example

Box Shadow Example

Code:

.shadow-text {
  font-size: 2rem;
  text-shadow: 2px 2px 4px #000000;
}
.shadow-box {
  box-shadow: 2px 2px 4px #000000;
}

7. CSS Multiple Columns

CSS Multiple Columns is a property that allows you to divide content into multiple columns, similar to how text is formatted in newspapers or magazines. It provides properties such as column-count, column-gap, and column-rule, which enable developers to control the number of columns, the spacing between them, and the visual separation between columns. This feature is particularly useful for improving readability and creating visually appealing layouts for large blocks of text.

8. CSS Table

CSS Table is a property that allows you to style HTML tables to enhance their appearance and improve readability. It includes properties such as border-collapse, border-spacing, and various styling options for table headers, rows, and cells. By using CSS Table properties, developers can create visually appealing tables that effectively present data while maintaining a consistent design with the rest of the webpage.

9. CSS Lists

CSS Lists is a property that allows you to style HTML lists, including ordered lists (ol) and unordered lists (ul). It provides properties such as list-style-type, list-style-image, and list-style-position, which enable developers to customize the appearance of list items, including bullet styles, numbering formats, and the position of markers. By using CSS Lists properties, developers can create visually appealing and well-structured lists that enhance the overall design of a webpage.

10. CSS Transforms

CSS Transforms is a property that allows you to apply various transformations to elements on a webpage, such as translation, rotation, scaling, and skewing. It provides properties like transform and transform-origin, which enable developers to manipulate the position, size, and orientation of elements in a 2D or 3D space. By using CSS Transforms, developers can create dynamic and interactive effects that enhance the user experience and visual appeal of a webpage.

Translate
Rotate
Scale
Skew

Code:

.trans-box {
  width: 150px;
  height: 150px;
  background-color: red;
  color: white;
  display: inline-flex;
  justify-content: center;
  align-items: center;
  margin: 20px;
  transition: transform 1s;
}
.tr-translate:hover {
  transform: translate(20px, -20px);
}
.tr-rotate:hover {
  transform: rotate(45deg);
}
.tr-scale:hover {
  transform: scale(1.5);
}
.tr-skew:hover {
  transform: skewX(20deg);
}
