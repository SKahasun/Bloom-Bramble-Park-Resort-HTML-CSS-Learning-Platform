2. Input Types

HTML5 introduced several new input types that provide better user experience and validation. For example, <input type="email"> is used for email addresses, <input type="date"> for date selection, and <input type="range"> for numeric ranges. These input types are more user-friendly and provide built-in validation capabilities.

Code: 

text, password, email, number, tel, url, search,
date, datetime-local, month, week, time,
radio, checkbox,
file, hidden,
range, color,
submit, reset, button, image
