1. Formatting Elements

Formatting elements are used to format text in HTML. They include headings, paragraphs, hyperlinks, images, lists, tables, and more.
example:

Bold Text

This is bold text

Italic Text

This is italic text

Underline Text

This is Underline text

Underline Text by ins

This is Underline text with ins

Delete Text by ins

The TajMahal is located in Bombay Agra

Highlighted text

This is Highlighted text

Superscript Text

(a + b)2 = a2 + 2ab + b2


Subscript Text

H2O


Code: 

<hr />

<h2>Bold Text</h2>
<b>This is bold text</b>
<br />

<h2>Italic Text</h2>
<i>This is italic text</i>
<br />

<h2>Underline Text</h2>
<u>This is Underline text</u>
<br />

<h2>Underline Text by ins</h2>
<u>This is Underline text with ins</u>
<br />

<h2>Delete Text by ins</h2>

<h2>Highlighted text</h2>

<br />

2. HTML Style Attributes

The style attribute in HTML is used to apply inline CSS styles to an element. It allows you to specify various CSS properties directly within the HTML tag. The syntax for using the style attribute is as follows:

Hello, World!

This is a paragraph with inline styles.

Code: 

<h2 style="color: blue; text-align: center">Hello, World!</h2>
<p style="font-size: 20px; color: green">
  This is a paragraph with inline styles.
</p>
