1. CSS Variables

CSS variables, also known as custom properties, allow you to store values that can be reused throughout your CSS. They are defined using the -- prefix and can be accessed using the var() function. For example, you can define a primary color variable and use it for multiple elements, ensuring consistency and making it easier to update the color scheme of your website. CSS variables can also be dynamic, allowing you to change their values based on user interactions or media queries.

Standard Account Card


This card uses `var(--primary-brand)` for its border and text color, and `var(--card-padding)` for spacing.


If we change `--primary-brand` in the CSS file, every single card on the website updates instantly.

Code:

:root {
  --primary-brand: red;
  --secondary-brand: #0c0c0c;
  --alert-color: red;
  --text-muted: #895353;
  --card-padding: 20px;
  --border-radius-md: 8px;
}
.theme-card {
  background-color: white;
  border: 1px solid var(--primary-brand);
  border-radius: var(--border-radius-md);
  padding: var(--card-padding);
  color: var(--primary-brand);
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
.theme-btn {
  background-color: var(--primary-brand);
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 8px;
  cursor: pointer;
}

2. CSS Calculation

The calc() function in CSS allows you to perform calculations to determine CSS property values. This is particularly useful for responsive design, where you may want to set widths, heights, or other properties based on a combination of fixed and relative units. For example, you can use calc(100% - 200px) to create a layout where an element takes up the remaining space after accounting for a fixed width sidebar. The calc() function can handle addition (+), subtraction (-), multiplication (*), and division (/) operations, making it a powerful tool for dynamic styling.


Sidebar
Width: 200px (Fixed)
Main Content Area
Width: calc(100% - 200px)

This area automatically fills the remaining space, ensuring the layout never breaks regardless of screen size.

Code:

.calc-container {
  height: 200px;
  border: 1px dashed rgb(209, 13, 13);
  background-color: var(--primary-brand);
  position: relative;
}
.calc-sidebar {
  width: 200px;
  height: 100%;
  background-color: #fededf;
  float: left;
  padding: 10px;
  box-sizing: border-box;
}
.calc-content {
  width: calc(100% - 200px);
  height: 100%;
  background-color: white;
  float: left;
  padding: 20px;
  box-sizing: border-box;
  border-left: 1px solid #ccc;
}

3. CSS Counters

CSS counters are a feature that allows you to automatically number elements in a document. They are particularly useful for creating numbered lists, section headings, and other content that needs to be numbered. Counters are initialized using the `counter-reset` property and incremented using the `counter-increment` property. The value of a counter can be displayed using the `content` property with the `counter()` function. For example, you can create a legal document with sections and clauses that are automatically numbered based on their hierarchy. This helps maintain consistency and saves time when adding or rearranging content.

Code:

.legal-doc {
  counter-reset: section;
  background-color: #fff;
  padding: 20px;
  border: 1px solid red;
}
.legal-doc h4 {
  counter-reset: clause;
  color: red;
  margin-top: 20px;
}
.legal-doc h4::before {
  counter-increment: section;
  content: "Section " counter(section) ": ";
}
.legal-doc p.clause::before {
  counter-increment: clause;
  content: counter(section) "." counter(clause) " ";
  font-weight: bold;
  color: white;
}
