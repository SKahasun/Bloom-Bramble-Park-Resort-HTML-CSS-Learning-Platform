1. CSS Units

CSS units are used to specify lengths, sizes, and other measurements in CSS. They can be broadly categorized into absolute and relative units.

CSS Units

Reletive Length

em

Usling font-size=30px current element

Using font-size=2em

This is a paragraph which location is also in parent(div). Font size of this paragraph in 2em but actual font size of this paragrph (30*2=60px) because em means two times of the sixe of the current element's font

Rem

Usling font-size=30px in parents

This is a paragraph which location is parent. Font size of this paragraph in 30px

Using font-size=2rem in child

This is a paragraph which location is child. Font size of this paragraph in 2em but actual font size of this paragrph (30*2=60px) because em means two times of the sixe of the current element's font

  font-size: 30px;
}
.custom {
  font-size: 2em;
}
.parents {
  font-size: 30px;

  font-size: 2rem;

2. Shorthand Properties

Shorthand properties in CSS allow you to set multiple related properties in a single declaration. This can make your CSS more concise and easier to read. Some common shorthand properties include:

• Margin: The margin property can set the margin for all four sides of an element (top, right, bottom, left) in one declaration. For example: margin: 10px 20px 15px 5px; sets the top margin to 10px, right margin to 20px, bottom margin to 15px, and left margin to 5px.
• Padding: Similar to margin, the padding property can set the padding for all four sides of an element in one declaration. For example: padding: 10px 20px; sets the top and bottom padding to 10px and the left and right padding to 20px.
• Border: The border property can set the width, style, and color of an element's border in one declaration. For example: border: 2px solid red; sets a solid red border with a width of 2 pixels.
