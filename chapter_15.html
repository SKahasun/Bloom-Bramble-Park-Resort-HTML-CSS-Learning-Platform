1. CSS Backgrounds Properties

CSS provides several properties to control the background of an element. The <background-color> property allows you to set a solid color as the background, while the <background-image> property lets you use an image as the background. You can also control how the background image is displayed using properties like <background-repeat>, <background-position>, and <background-size>. These properties give you flexibility in designing the look and feel of your webpage.

2. CSS Gradient Backgrounds

CSS gradients allow you to create smooth transitions between two or more colors. There are two main types of gradients: linear and radial. A linear gradient transitions colors along a straight line, while a radial gradient transitions colors outward from a central point. You can customize the direction, shape, and color stops of gradients to create visually appealing backgrounds for your web pages.

Code:

      .linear-down {
        background-image: linear-gradient(red, yellow);
        color: white;
        height: 100px;
      }

      .linear-right {
        background-image: linear-gradient(to right, red, yellow);
        color: white;
        height: 100px;
      }

      .linear-diag {
        background-image: linear-gradient(to bottom right, red, yellow);
        color: white;
        height: 100px;
      }

      .radial {
        background-image: radial-gradient(circle, red, yellow, green);
        height: 100px;
      }

      .repeat-linear {
        background-image: repeating-linear-gradient(
          45deg,
          red,
          red 10px,
          white 10px,
          white 20px
        );
        color: black;
        font-size: larger;
        height: 100px;
      }
