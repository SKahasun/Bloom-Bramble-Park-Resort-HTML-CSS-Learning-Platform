1. First Webpage

Welcome to my Webpage.

Code: 

<p>
  Welcome to my Webpage.
</p>

2. Character Encoding

Meta: UTF-8

Code: 

<meta charset="UTF-8" />

3. Element Attributes

Attributes provide additional information about an element. They are always included in the opening tag and usually come in name/value pairs like name="value".

Code: 

<img src="image.jpg" alt="Description of image" />

4. Favorite Icon

The favorite icon (favicon) is a small image displayed in the browser tab.

Code: 

<link rel="icon" type="image/x-icon" href="/favicon.ico" />

5. Comments

Comments are used to explain code and are not displayed in the browser.

Code: 

<!-- This is a comment -->
