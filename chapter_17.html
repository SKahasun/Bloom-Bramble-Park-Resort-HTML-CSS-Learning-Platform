1. CSS Position

CSS position property is used to specify the positioning of an element on a webpage. It allows you to control how an element is positioned in relation to its normal flow, its parent container, or the viewport. The different values for the position property include static, relative, absolute, fixed, and sticky. Each value has its own behavior and use cases, allowing developers to create complex layouts and designs.

2. CSS Float

CSS float property is used to position an element to the left or right of its container, allowing other elements to flow around it. The float property can take values like left, right, none, and inherit. When an element is floated, it is taken out of the normal document flow and shifted to the specified side of its container. This behavior is useful for creating layouts with multiple columns or for aligning images within text content.

Float left

Float right

Code:

.float {
  border: 3px dotted red;
  width: 400px;
  height: 50px;
}
.floatLeft {
  border: 1px solid red;
  margin: 2px;
  float: left;
}
.floatRight {
  border: 1px solid red;
  margin: 2px;
  float: right;
}

3. CSS Flexbox

CSS Flexbox is a layout model that allows for flexible and responsive design of web pages. It provides an efficient way to distribute space and align items within a container, even when their size is unknown or dynamic. The flex container's children (flex items) can be aligned in various ways, such as along the main axis or cross axis, and can be distributed with different spacing rules.

4. CSS Grid

CSS Grid is a powerful layout system that allows developers to create complex and responsive web designs with ease. It provides a two-dimensional grid-based layout system, enabling the arrangement of elements in rows and columns. With CSS Grid, you can define grid containers and grid items, and specify how they should be positioned and sized within the grid. This makes it easier to create dynamic layouts that adapt to different screen sizes and devices.

5. CSS Alignment

CSS alignment properties are used to control the positioning and alignment of elements within a container. These properties include justify-items, align-items, justify-content, and align-content. They allow developers to specify how items should be aligned along the main axis and cross axis of a container, as well as how space should be distributed between items. Proper use of CSS alignment can enhance the visual appeal and usability of a web page by ensuring that elements are positioned in a way that is visually pleasing and easy to navigate.

Start
End
Center
Space Between

Code:

.justifyContent-container {
  display: grid;
  grid-template-columns: auto auto auto;
  gap: 10px;
  border: 2px solid red;
  width: 450px;
  height: 300px;
  padding: 10px;
}

  background-color: red(0, 187, 250);
  padding: 10px;
  border: 1px solid rgb(77, 40, 40);
}
.start {
  justify-items: start;
  align-items: start;
  justify-content: start;
}
.end {
  justify-items: end;
  align-items: end;
  justify-content: end;
}
.center {
  justify-items: center;
  align-items: center;
  justify-content: center;
}
.space-Between {
  justify-items: center;
  align-items: center;
  justify-content: space-between;
}
