1. Data URLs

Data URLs allow you to embed small files directly within your HTML or CSS code. This can be useful for including images, fonts, or other resources without needing separate files. A data URL consists of a prefix (data:), followed by the MIME type of the data, and then the actual data encoded in base64.

2. Block Elements

Block elements are HTML elements that take up the full width available and start on a new line. Common block elements include <div>, <p>, <h1> to <h6>, <ul>, <ol>, and <li>. They are typically used to structure the content of a webpage.

3. Inline Elements

Inline elements are HTML elements that do not start on a new line and only take up as much width as necessary. Common inline elements include <span>, <a>, <strong>, <em>, and <img>. They are typically used to style or link small portions of text within block elements.
This is an inline element example using the <strong> tag to emphasize text.

Inline Elements



Code: 

<div id="inlineElements" style="display: inline;">
  <h1>Inline Elements</h1><br><br>
  <img src="Image/cup.jpg" width="800" alt="Coffee" />
</div>

4. Inline Block Elements

Inline-block elements are a hybrid between block and inline elements. They flow with the text like inline elements but can have width and height set like block elements. Common inline-block elements include <img> and <button>. They are useful for creating layouts where elements need to be aligned horizontally but still maintain block-like properties.
Here are some headings styled as inline-block elements:

My First h1

My Second h2

My Fourth h4

My Fifth h5
My Sixth h6

Code: 

<h1 style="display: inline-block; margin-right: 10px;">My First h1</h1>
<h2 style="display: inline-block; margin-right: 10px;">My Second h2</h2>
<h4 style="display: inline-block; margin-right: 10px;">My Fourth h4</h4>
<h5 style="display: inline-block; margin-right: 10px;">My Fifth h5</h5>
<h6 style="display: inline-block; margin-right: 10px;">My Sixth h6</h6>
